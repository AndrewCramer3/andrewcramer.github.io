Project Dad Bod

About the Game

Project Dad Bod is a game made in Unreal Engine 5.1 using blueprints that was worked on by a group of five over the course of a semester. It is a Tower Defense game where you defend against a horde of angry drunk dads and in between waves, you scavenge for materials to upgrade your base and place traps.

What I've Learned

This taught me a lot of how to scope a project. We scoped this a little too big, which is why the presentation is a little rough. But We got a sloppy build of the game with all the features we wanted by the end of the semester, which was exciting and pleasing to see. Another thing this taught me is how to work in a group. I worked in groups for Game Jams, but this was over an entire semester. It taught me to always figure out what we want out of a feature before it is programmed, and to set realistic deadlines for features.
