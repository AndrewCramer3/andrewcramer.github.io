Mimic Hunter

About the Game

Mimic Hunter is a Game Jam project I worked on with a group of seven, and is a Bullet Hell / Hide and Seek project that is about finding Mimics and slaying them before you die to their wrath. This project was made in Godot of September 2022.

My Role / Contributions

  • AI

What I've Learned

This was one of my first game jams and group projects so interacting with others in a group setting was something I had to figure out. Others within the group helped me understand and learn things about Godot, since I had only used Game Engines a handful of times up to this point. I also learned a lot about how art assets are created and used within the project. Overall, this experience taught me a lot about how game engines are structured, some of the basic principles when making systems for features and a lot of other things.
